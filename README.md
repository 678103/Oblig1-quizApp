Har laget en app hvor brukeren kan spille quiz (Quiz) og legge inn bilder (Gallery).

Quiz:
I quiz-spillet telles det poeng for hvert riktig svar. Hvis bruker svarer feil så vil scoren holde seg på samme tall. 
Bruker får også beskjed om svaret var rett eller feil.

Layout:
Brukeren kan legge inn valgfrie bilder fra sin mobil. Det vil alltid ligge tre bilder der uansett, til bruk i quizen. 
Bruker kan sortere bildene sine i alfabetisk rekkefølge.
